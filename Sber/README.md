|   a   |   b   |
|-------|-------|
| null  | null  |
|   1   |   2   |
| null  |   1   |
|   3   | null  |
|   3   |   5   |

```sql
select count(*) c1, count(1) c2, count(a) c3, count('a') c4, count(distinct a) c5 from table
```
